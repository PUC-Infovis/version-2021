Cascading Style Sheets

Cascading Style Sheets


  • Languaje de hojas de estilo.
  • Permiten declarar propiedades visuales para HTML.
  • El navegador web es quien interpreta dichas declaraciones.

						p {
						  color: blue;
						}

						selector {

						}
