seleccion.data


  • Nuevos datos aparecen ➡️ enter
  • Datos cambian ➡️ update
  • Elementos desaparecen ➡️ exit



Método join para flujo usual de data join



            svg
              .selectAll("rect")
              .data(datos)
              .join("rect");
          

            svg
              .selectAll("rect")
              .data(datos)
              .join(
                enter => enter.append("rect"),
                update => update,
                exit => exit.remove()
              );
